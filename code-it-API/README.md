## API for Code-It

Includes:

- Login/Sign Up API with JWT tokens (needs to add whitelist)
- Protected endpoint
- Searching for users

Requires you to add a .env file for the database credentials!

Should contain the following information:
DB_HOST = **\*
DB_USER = \*\*\*\***
DB_PASSWORD = \*\*\*\*
DB_NAME = \*\*\*

TOKEN_SECRET = **\*\***

Run `npm start` to start the server.
